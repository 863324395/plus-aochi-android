# app module

结构目录：
```shell
--base
    ...
--config
--datasource
    --local
    --remote
--modules
    ...
```
项目主工程，实现ts的所有功能模块

###base目录：
    ts的基类，继承自common包下的base基类

###config目录：
    ts中的一些配置参数

###data.source目录：
    数据源管理
 - local目录
 - remote目录

###modules目录
    ts项目的功能模块
